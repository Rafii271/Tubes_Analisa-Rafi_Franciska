# Tubes_Analisa-Rafi_Franciska
# Tugas Besar Analisa Keputusan

## Deskripsi
Notebook ini berisi implementasi machine learning untuk memprediksi risiko gagal bayar (default) pada data P2P Lending menggunakan Random Forest.

## Dataset
Dataset diperoleh dari Kaggle:
https://www.kaggle.com/datasets/skhikingkevin/online-p2p-lending

Dataset diunduh langsung di Google Colab menggunakan KaggleHub.

## Metode
- Random Forest Classifier
- Data imbalance handling
- Train-test split 70:30
- Evaluasi: Confusion Matrix, Classification Report, ROC-AUC
- Hyperparameter tuning (6 skenario)

## Cara Menjalankan
Klik tombol **Open in Colab** pada file notebook.
